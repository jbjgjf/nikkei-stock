# Rename Plan

## Naming Rules
- Datasets (csv): `data/<layer>/<domain>/<subject>[_<scope>][_YYYYMMDD].csv` using lower_snake_case.
- Scripts (py): `scripts/<phase>_<verb>_<object>[_<qualifier>].py` using lower_snake_case.
- Figures (png/pdf): `reports/figures/<phase>/<topic>__<metric>__<YYYYMMDD>.<ext>` using lower_snake_case.

## Folder Structure Recommendation
- data/raw
- data/interim
- data/features
- data/scores
- reports
- scripts
- docs

## Phase Mapping
- P0: External/source ingestion (JPX, GX League, EEGS indices, TCFD/SBTi lists).
- P1: Entity mapping, normalization, universe construction, name mapping.
- P2: Financial data acquisition and shaping (yfinance outputs).
- P2_75: 2.5 screening outputs (screened/excluded + summaries).
- P3: EEGS linking, graph extraction, emissions trend analysis.

## Execution Plan
1) Apply folder structure and move raw inputs to `data/raw`.
2) Move derived datasets into `data/interim`, `data/features`, and `data/scores`.
3) Rename scripts to match phase+verb naming and update imports/calls.
4) Update all hardcoded paths listed in `docs/PATHS_TO_UPDATE.md`.
5) Validate by re-running the pipelines for P1->P3 outputs.
